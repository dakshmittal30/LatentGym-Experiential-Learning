What's in the framework

  • 7 environments built on TextArena, each composable as core × latent × prompt × feedback × N.
  • Evaluation pipeline for single-agent and double-agent (agent-switching) runs, plus exploration/exploitation metrics.
  • Cross-Task RL training via GRPO over full task sequences, with an integrated dashboard for inspecting runs.
  • Recorded trajectories for frontier models and Qwen3-8B (base / single-task RL / cross-task RL), powering this site's explorer.

Add a new environment

An environment is defined by a handful of small modules. Implement each interface and register it — the evaluator, trainer, and dashboards pick it up automatically.

ModuleResponsibility
core_env.pyThe single-episode decision problem (states, actions, reward).
latents.pyLatent families — generator-based or filter-based.
prompts.pyThe no/some/full info prompt templates.
feedbacks.pystandard vs information inter-task feedback.
trajectory_generator.pyRoll out a full N-task sequence.
